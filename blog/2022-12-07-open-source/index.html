I planned to write this post way earlier, and I will extend it from time to time.

I have benefited so much from free software I just want to share why.

My personal perception of software is either "It works as expected" or "it has bugs and flaws". So if a software behaves as the user expected it to, that's almost the best case scenario, which is rarely even appreciated.

So I want to give examples, which you may have experienced yourself, where software does not do what the user expected. And I want to show how it would make a difference, if these projects happened to be open source.

In the second part of this post I want to share the power of software which works even without an internet connection.
